## Installation

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.0.3.

```bash
cd web-spa
pnpm i
```

## Running the app

```bash
# development
pnpm start
```

## Test

```bash
# unit tests
pnpm test
```
